Get the DeepInfer app

  1. Open 3D Slicer. Once it is launched click on the "Extensions Manager" button.




  2. Once the Extension Manager pops up, make sure that the "Install Extensions" tab is selected. Click on the search box on the top right and type DeepInfer.



  3. Click on the install button and wait until it installation is done. Then press the restart button.

Validate your installation

Once you installed DeepInfer, follow the steps below to make sure that the DeepInfer is installed correctly and can communicate with the Docker application.

  1. Go to the modules menu and choose Machine Learning > DeepInfer module.




  2. In the DeepInfer module user interface, find the docker settings tab, press the test button to test docker configuration.


    If the Docker is installed and configured correctly on your system you will get the following popup widow with information about the version of Docker. Otherwise, if you are getting a warning for docker configuration check your Docker installation and configuration.
