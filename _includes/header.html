DeepInfer

Deep Learning Deployment Toolkit for Medical Imaging

About DeepInfer

DeepInfer is an open-source toolkit for medical image analysis and processing with deep learning models. DeepInfer seamlessly integrates deep learning technology into the existing medical image analysis platforms such as 3D Slicer. It allows clinical researchers and biomedical engineers to deploy multi-platform pre-trained deep learning models and use task-specific deep models, without the need for further software development and configuration.
